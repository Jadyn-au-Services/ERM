| Command            | Explanation                           |
| ------------------ | ------------------------------------- |
| `/erlc link`       | Link your ER:LC server with ERM!           |
| `/erlc info`       | View info about your current players in-game. <Frame><img src="../images/api/commandlist/erlcinfo.png" /></Frame>  |
| `/erlc kills`      | See the kill logs of your server! <Frame><img src="../images/api/commandlist/erlckills.png" /></Frame> |
| `/erlc players`    | See all players in the server. <Frame><img src="../images/api/commandlist/erlcplayers.png" /></Frame> |
| `/erlc bans`       | Filter the bans of your server. (With this command, you can see if a player is banned or not.) <Frame><img src="../images/api/commandlist/erlcbans.png" /></Frame> |
| `/erlc command`    | Whether the user joined the community <Frame><img src="../images/api/commandlist/erlccommand.png" /></Frame> <Frame><img src="../images/api/commandlist/erlccommand2.png" /></Frame> |
| `/erlc hint`       | Run an ER:LC hint message. |
| `/erlc message`    | Run an ER:LC server accouncement. |
| `/erlc logs`       | View the ER:LC server command logs for your server. <Frame><img src="../images/api/commandlist/erlclogs.png" /></Frame> |
| `/erlc playerlogs` | View your ER:LC join and leave logs. <Frame><img src="../images/api/commandlist/erlcplayerlogs.png" /></Frame> |
