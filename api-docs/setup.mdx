    <Accordion
        title="1 | Join the ERLC Server"
        icon="person-to-portal"
    >
    1. Join the ER:LC server and open the server settings.
    .2 Find the "ER:LC API" section, and click edit on the "Server key" tab.

    <Frame>
    <img src="../images/api/setup/apikey.png" />
    </Frame>

    3. Copy the API key, and then continue to step 2!

        
    </Accordion>
    <Accordion
        title="2 | Link the key with ERM"
        icon="link"
    >
    1. Once you have the API key copied, you are going to go to the Discord server and run the command /erlc link

    2. input the key in the space provided within the link command, and send the command. As long as the API key is valid, you should see a message like this:
    <Frame>
    <img src="../images/api/setup/erlclink.png" />
    </Frame>


    <Tip>If you do not see this message, tell them to regenerate the key and try again. If it still does not work, contact one of the management members.</Tip>
        
    </Accordion>
